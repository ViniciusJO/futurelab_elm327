# Sistema de coleta

Persistencia e visualizacao de dados de GPS e OBD2

Instalacao:

```sh
git clone https://github.com/ViniciusJO/futurelab_elm327.git

cd futurelab_elm327/client
npm install

cd ../server
npm install

npm run build-client
npm run build
```

Inicializacao:

```sh
npm run start
```
